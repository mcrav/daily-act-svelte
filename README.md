# sv

Demo: https://daily-act-svelte.vercel.app

SvelteKit app for practising acceptance and commitment therapy (ACT) and mindfulness exercises using an AI chatbot.

## Installation

`npm i`

## Run locally

`npm run dev`

## Build

`npm run build`

## Technologies used

- SvelteKit
- Langchain
- OpenAI API
- Tailwind
- Skeleton UI
- XState
- Vercel
